1. Опис проєкту

1.1. Цільова аудиторія

Учні 10–11 класів загальноосвітніх шкіл (при вивченні розділів стереометрії), а також студенти вищих навчальних закладів математичних спеціальностей.

1.2. Об'єкти дослідження

Геометричні тіла та многогранники, параметри яких динамічно змінюються користувачем:

  • Правильні піраміди (динамічна зміна радіуса основи, висоти та кількості бічних граней від 3 до 12).
  • Правильні призми / Циліндри (параметризація висоти, радіуса основи та сегментації поверхонь).

2. Функціональність додатку

Функція Опис Реалізація
Конструктор призм та пірамід Візуалізація 3D-геометрії та плавне перемикання між типами тіл. Компоненти геометрії a-entity в A-Frame.
Динамічний UI-інтерфейс Зміна висоти фігури, радіуса та кількості граней за допомогою повзунків. HTML5 Range Sliders та JavaScript логіка зв'язування.
Реальний перерахунок (STEM) Автоматичний розрахунок площі основи та об'єму фігури при будь-якій зміні її форми. Вбудований JavaScript-калькулятор (математичний процесор).
ML Голосове керування Перебудова 3D-сцени та зміна розмірів тіл за допомогою голосових команд користувача. Інтеграція машинного розпізнавання мови Web Speech API.
Інтерактивні підказки Інформування користувача про стан мікрофона та поточні розпізнані команди. Динамічні текстові блоки індикації статусу на UI-панелі.

Згідно з методологічними рекомендаціями, архітектура вебресурсу має наступну ієрархічну структуру папок:

educational-ar-project/

└── docs/                     # Посібник користувача та матеріали захисту

5. Переваги проєкту

  • STEM-ориєнтованість: Поєднання геометрії з алгоритмізацією та технологіями ШІ.
  • Миттєва доступність: Запуск в один клік без встановлення стороннього софту.
  • Інклюзивність: Голосове керування демонструє роботу Machine Learning та допомагає учням з різними модальностями сприйняття.

6. Обмеження та майбутні покращення

Поточні обмеження: Залежність від мережі Інтернет для завантаження бібліотек; стабільність розпізнавання голосу залежить від рівня стороннього шуму.

Планові покращення: Впровадження інструменту для побудови інтерактивних січних площин (перерізів многогранників) та експорт звітів у Excel.

7. Інструкція з використання голосового ШІ

  1. Натиснути кнопку «Запустити геометричну майстерню».
  2. Натиснути кнопку «Увімкнути голос» на екрані 3D-сцени.
  3. Надати браузеру допуск до мікрофона.
  4. Чітко вимовити команду українською мовою: «призма», «піраміда» або «вище».
