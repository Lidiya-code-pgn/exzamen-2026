# 🎓 Віртуальна геометрична майстерня

Інноваційний освітній AR/VR проєкт із геометрії (стереометрії) для учнів 10–11 класів та студентів математичних спеціальностей.

## 👤 Автор проєкту
* **Студентка:** Шевцова Лідія Вікторівна
* **Група:** МІМ-25
* **Спеціальність:** Середня освіта (Математика та інформатика)
* **Дисципліна:** Інноваційні цифрові технології в освіті
* **Рік:** 2026

## 🚀 Посилання на вебресурс
* 🌍 **Головна сторінка проєкту:** [Запустити вебресурс](https://lidiya-code-pgn.github.io/CHLV-26/)
* 🛠 **Прямий запуск 3D-майстерні:** [Відкрити інтерактивну сцену](https://lidiya-code-pgn.github.io/CHLV-26/workshop.html)

## ⚙️ Технологічний стек
* **3D-графіка:** A-Frame (WebVR), Three.js, WebGL
* **Логіка та розрахунки:** JavaScript (ES6+ калькулятор об'єму та площі)
* **Штучний інтелект (ML):** Web Speech API (модуль голосового керування українською мовою)
* **Інтерфейс:** HTML5, CSS3
